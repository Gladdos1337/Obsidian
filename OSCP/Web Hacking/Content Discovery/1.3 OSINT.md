**Wappalyzer**

Wappalyzer ([https://www.wappalyzer.com/ (opens in new tab)](https://www.wappalyzer.com/)) is an online tool and browser extension that helps identify what technologies a website uses, such as frameworks, Content Management Systems (), payment processors and much more, and it can even find version numbers as well.

**Github**

When users have finished making their changes, they commit them with a message and then push them back to a central location (repository) for the other users to then pull those changes to their local machines. GitHub is a hosted version of on the internet. Repositories can either be set to public or private and have various access controls. You can use GitHub's search feature to look for company names or website names to try and locate repositories belonging to your target. Once discovered, you may have access to source code, passwords or other content that you hadn't yet found.

**Buckets**

S3 Buckets are a storage service provided by Amazon , allowing people to save files and even static website content in the cloud accessible over and HTTPS. The owner of the files can set access permissions to either make files public, private and even writable. Sometimes these access permissions are incorrectly set and inadvertently allow access to files that shouldn't be available to the public. The format of the buckets is (s)://**{name}.**[](http://s3.amazonaws.com/)[**.amazonaws.com** (opens in new tab)](http://s3.amazonaws.com/) where {name} is decided by the owner, such as [tryhackme-assets.](http://tryhackme-assets.s3.amazonaws.com)[.amazonaws.com (opens in new tab)](http://tryhackme-assets.s3.amazonaws.com). buckets can be discovered in many ways, such as finding the URLs in the website's page source, GitHub repositories, or even automating the process. One common automation method is by using the company name followed by common terms such as **{name}**-assets, **{name}**-www, **{name}**-public, **{name}**-private, etc.
