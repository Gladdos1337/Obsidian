## Notes

- **Priority order**: Linux PrivEsc → Web Exploitation → Windows PrivEsc → Active Directory → BOF
- **Rooms by Tib3rius** (Linux PrivEsc, Windows PrivEsc) are particularly OSCP-aligned
- **Wreath** is the best pivoting/AD chain room on THM — save it for when you're comfortable
- **Buffer overflow** rooms matter less now (OSCP 2023+ reduced BOF weight) but still good to know
- Complement THM with **Proving Grounds Practice** (OffSec) when you feel ready for harder boxes
