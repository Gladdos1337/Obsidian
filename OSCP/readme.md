# CPTS / OSCP Study Vault

**Last restructured:** June 2026  
**Goal:** Pass the HTB Certified Penetration Testing Specialist (CPTS) exam

## Vault Structure

This vault is organized by the full CPTS penetration testing methodology:

- **00 - Methodology and Workflow** — Step-by-step methodology, exam checklist, quick reference card
- **01 - Information Gathering** — Nmap mastery, passive recon, active recon, DNS enumeration, service enumeration
- **02 - Vulnerability Assessment** — Nessus, OpenVAS, searchsploit, CVSS
- **03 - Web Application Hacking** — SQLi, XSS, LFI/RFI, SSRF, Command Injection, File Upload, Auth Bypass, API, CMS
- **04 - Password Attacks** — Online brute force, offline cracking, hashcat reference, Kerberos attacks
- **05 - Network Attacks** — Sniffing/spoofing, firewall evasion, pivoting & tunneling
- **06 - Shells and Payloads** — Reverse shells, payload generation (MSFVenom), file transfers
- **07 - Metasploit Framework** — Complete Metasploit reference
- **08 - Linux Privilege Escalation** — Comprehensive Linux PE
- **09 - Windows Privilege Escalation** — Comprehensive Windows PE
- **10 - Active Directory** — Recon, attack vectors, lateral movement, ACL abuse, domain dominance
- **11 - Post Exploitation** — Linux and Windows post-exploitation
- **12 - Pivoting and Port Forwarding** — SSH tunneling, Chisel, Ligolo-ng
- **13 - Report Writing** — Professional pentest report template
- **14 - Common Ports and Services** — Full port reference table
- **Boxes** — Writeups for HTB/THM boxes
- **CTF Writeups** — CTF challenge notes

## How to Use

1. Start with the **CPTS Master Index** for an overview of everything
2. Follow the **CPTS Methodology** as your exam workflow guide
3. Use the **Quick Reference Card** for emergency cheat-sheet access
4. Search tools by name or navigate by the numbered folder structure
